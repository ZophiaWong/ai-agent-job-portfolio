# 04_二叉树

二叉树题核心是递归定义。面试中先说清楚当前节点需要从左右子树拿到什么，再决定是前序、后序、层序还是中序。

## 面试识别信号

- 题目出现 depth、height、path、ancestor、level order、valid BST。
- 每个节点的答案依赖左右子树：优先 DFS。
- 需要按层输出、最短层数、右视图：优先 BFS。
- BST 题通常利用中序有序或上下界约束。
- 公共祖先题常用“当前节点是否命中目标，左右子树是否返回目标”。

## 核心套路

- DFS 递归：定义函数返回值，而不是先急着写遍历。
- 前序适合自顶向下传状态；后序适合自底向上汇总结果。
- BFS 用 `deque`，按层处理时记录当前队列长度。
- BST 验证可以传 `(low, high)`，避免只比较父节点的局部错误。
- LCA 中，左右子树都找到目标时，当前节点就是答案。

## Python 模板

```python
from collections import deque
from typing import Optional


class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right


def dfs(root: Optional[TreeNode]) -> int:
    if not root:
        return 0
    left = dfs(root.left)
    right = dfs(root.right)
    return max(left, right) + 1
```

```python
def level_order(root: Optional[TreeNode]):
    if not root:
        return []

    result = []
    queue = deque([root])
    while queue:
        level = []
        for _ in range(len(queue)):
            node = queue.popleft()
            level.append(node.val)
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        result.append(level)
    return result
```

## 讲解例题

### LeetCode 104. Maximum Depth of Binary Tree

思路：一棵树的最大深度等于左右子树最大深度的较大值加 1。空节点深度为 0。

```python
from typing import Optional


class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if not root:
            return 0
        return max(self.maxDepth(root.left), self.maxDepth(root.right)) + 1
```

复杂度：时间 `O(n)`，递归栈空间 `O(h)`。

### LeetCode 236. Lowest Common Ancestor of a Binary Tree

思路：如果当前节点是 `p` 或 `q`，返回当前节点。递归查左右子树；如果左右都非空，说明两个目标分布在两侧，当前节点就是最近公共祖先。

```python
from typing import Optional


class Solution:
    def lowestCommonAncestor(
        self,
        root: Optional[TreeNode],
        p: TreeNode,
        q: TreeNode,
    ) -> Optional[TreeNode]:
        if not root or root is p or root is q:
            return root

        left = self.lowestCommonAncestor(root.left, p, q)
        right = self.lowestCommonAncestor(root.right, p, q)

        if left and right:
            return root
        return left or right
```

复杂度：时间 `O(n)`，递归栈空间 `O(h)`。

## 常见坑

- 递归函数的返回值要和全局答案区分清楚。
- 路径题要确认是根到叶、任意节点到任意节点，还是根到任意节点。
- BST 不能只检查当前节点和左右孩子，要检查整个上下界。
- 层序遍历按层输出时，要先固定 `len(queue)`。
- 递归深度在极端链状树上可能达到 `O(n)`。
