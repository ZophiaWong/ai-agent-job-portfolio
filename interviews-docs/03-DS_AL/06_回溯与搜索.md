# 06_回溯与搜索

本章覆盖回溯、DFS/BFS、图遍历、拓扑排序和连通分量。面试中要先判断是在枚举所有解，还是在图/网格中寻找可达状态。

## 面试识别信号

- 题目要求列出所有组合、排列、子集：优先回溯。
- 题目要求在网格中找岛屿、路径、扩散：优先 DFS/BFS。
- 题目包含课程依赖、先后顺序：考虑拓扑排序。
- 题目问连通块数量、是否可达、最短步数：考虑图遍历。
- 题目需要剪枝：先排序、去重、提前返回不可能分支。

## 核心套路

- 回溯三件事：选择、递归、撤销选择。
- 组合问题通常传 `start`，避免重复选择。
- 排列问题通常用 `used`，表示某个元素是否已在路径中。
- BFS 适合无权图最短步数；DFS 适合连通块和完整搜索。
- 拓扑排序用入度表和队列，能处理完所有节点则无环。

## Python 模板

```python
def backtrack(nums):
    result = []
    path = []

    def dfs(start):
        result.append(path[:])
        for i in range(start, len(nums)):
            path.append(nums[i])
            dfs(i + 1)
            path.pop()

    dfs(0)
    return result
```

```python
from collections import deque


def bfs(graph, start):
    seen = {start}
    queue = deque([start])
    while queue:
        node = queue.popleft()
        for nxt in graph[node]:
            if nxt not in seen:
                seen.add(nxt)
                queue.append(nxt)
    return seen
```

## 讲解例题

### LeetCode 78. Subsets

思路：每个位置都可以选择加入或跳过。用 `start` 控制后续只能选择右侧元素，避免重复子集。

```python
from typing import List


class Solution:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        result = []
        path = []

        def dfs(start: int) -> None:
            result.append(path[:])
            for i in range(start, len(nums)):
                path.append(nums[i])
                dfs(i + 1)
                path.pop()

        dfs(0)
        return result
```

复杂度：共有 `2^n` 个子集，时间和输出空间都是 `O(n * 2^n)` 量级。

### LeetCode 200. Number of Islands

思路：扫描网格，遇到陆地就计数，并用 DFS 把同一岛屿的陆地全部标记为已访问。每个格子最多访问一次。

```python
from typing import List


class Solution:
    def numIslands(self, grid: List[List[str]]) -> int:
        if not grid or not grid[0]:
            return 0

        rows, cols = len(grid), len(grid[0])
        count = 0

        def dfs(r: int, c: int) -> None:
            if r < 0 or r >= rows or c < 0 or c >= cols:
                return
            if grid[r][c] != "1":
                return
            grid[r][c] = "0"
            dfs(r + 1, c)
            dfs(r - 1, c)
            dfs(r, c + 1)
            dfs(r, c - 1)

        for r in range(rows):
            for c in range(cols):
                if grid[r][c] == "1":
                    count += 1
                    dfs(r, c)

        return count
```

复杂度：时间 `O(mn)`，递归栈最坏 `O(mn)`。

## 常见坑

- 把 `path` 直接加入结果会被后续修改影响，必须用 `path[:]`。
- 组合和排列的去重方式不同，不要混用 `start` 和 `used`。
- 网格搜索要处理越界、访问标记和恢复时机。
- BFS 求最短步数时，一般按层推进，而不是只数访问节点。
- 拓扑排序最后要检查处理节点数是否等于总节点数。
- `numIslands` 会把 `grid` 中访问过的陆地改成 `"0"`；需保留输入时改用 `seen` 集合或深拷贝。大网格的递归 DFS 也可能超过 Python 递归深度，应换成显式栈。

## 面试表达

可以这样讲：如果题目要求枚举所有可能，我用回溯构造搜索树；如果题目问可达性或最短步数，我把状态看作图节点，再用 DFS 或 BFS 遍历。
