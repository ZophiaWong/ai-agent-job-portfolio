# 07_堆栈队列二分

本章覆盖栈、队列、堆和二分。它们都是“维护访问顺序或候选集合”的工具：栈处理最近未匹配，队列处理先进先出，堆处理动态极值，二分处理有单调性的搜索空间。

## 面试识别信号

- 括号匹配、路径简化、表达式：考虑栈。
- 层序遍历、最短步数、任务扩散：考虑队列 `deque`。
- Top K、动态最小/最大、合并多个有序流：考虑 `heapq`。
- 已排序数组、边界位置、最小可行值、最大可行值：考虑二分。
- 题目问“第 k 大/小”时，先判断能否用大小为 k 的堆。

## 核心套路

- 栈只看栈顶，适合处理最近一个未解决元素。
- 队列用 `collections.deque`，不要用 `list.pop(0)`。
- Python `heapq` 是小根堆；最大堆通常存负数。
- 二分要先确定搜索区间含义，再选择闭区间或半开区间。
- 对答案二分的关键是写出单调的 `can(x)` 判断函数。

## Python 模板

前提：`lower_bound` 只适用于升序 `nums`，并使用半开区间 `[left, right)`；若数组未排序、比较关系不单调，不能套用二分。`top_k_largest` 要求 `1 <= k <= len(nums)`。

```python
from collections import deque
import heapq


def queue_template(graph, start):
    queue = deque([start])
    seen = {start}
    while queue:
        node = queue.popleft()
        for nxt in graph[node]:
            if nxt not in seen:
                seen.add(nxt)
                queue.append(nxt)
```

```python
def lower_bound(nums, target):
    left, right = 0, len(nums)
    while left < right:
        mid = (left + right) // 2
        if nums[mid] < target:
            left = mid + 1
        else:
            right = mid
    return left
```

```python
def top_k_largest(nums, k):
    heap = []
    for x in nums:
        heapq.heappush(heap, x)
        if len(heap) > k:
            heapq.heappop(heap)
    return heap
```

## 讲解例题

### LeetCode 20. Valid Parentheses

思路：遇到左括号入栈；遇到右括号时，栈顶必须是对应左括号。最后栈为空才说明全部匹配。

```python
class Solution:
    def isValid(self, s: str) -> bool:
        pairs = {")": "(", "]": "[", "}": "{"}
        stack = []

        for ch in s:
            if ch in pairs.values():
                stack.append(ch)
            else:
                if not stack or stack[-1] != pairs[ch]:
                    return False
                stack.pop()

        return not stack
```

复杂度：时间 `O(n)`，空间 `O(n)`。

### LeetCode 215. Kth Largest Element in an Array

思路：维护大小为 `k` 的小根堆。堆里保留当前见过的最大 `k` 个数，堆顶就是第 `k` 大。

```python
import heapq
from typing import List


class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        heap = []
        for x in nums:
            heapq.heappush(heap, x)
            if len(heap) > k:
                heapq.heappop(heap)
        return heap[0]
```

复杂度：时间 `O(n log k)`，空间 `O(k)`。

## 常见坑

- `list.pop(0)` 是 `O(n)`，队列要用 `deque.popleft()`。
- `heapq` 默认小根堆，求最大值要取负数或维护大小为 k 的小根堆。
- 二分循环条件、边界更新和返回值必须和区间定义一致。
- 单调栈里存下标通常比存值更有用，因为答案常要求距离。
- Top K 题要先判断返回的是元素、频率还是顺序。
