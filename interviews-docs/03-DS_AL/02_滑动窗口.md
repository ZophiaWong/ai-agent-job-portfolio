# 02_滑动窗口

滑动窗口处理连续区间问题。它的本质是维护 `[left, right]` 内的状态，并在状态不合法或已经满足条件时移动左边界。

## 面试识别信号

- 题目要求连续子数组、连续子串、最长、最短、固定长度。
- 右边界每次向右扩展，窗口状态可以增量更新。
- 判断条件和窗口内部计数、和、不同元素数量有关。
- 需要返回长度、数量、是否存在，而不是返回所有组合。

## 核心套路

- 固定窗口：窗口大小固定为 `k`，每次加入右侧、移出左侧。
- 可变窗口：右指针扩张，条件不满足或过度满足时移动左指针。
- 最长类问题：通常在窗口合法时更新答案。
- 最短类问题：通常在窗口满足条件时尽量收缩。
- 计数类窗口要明确“窗口合法”的定义，例如缺少字符数、重复字符数、不同字符数。

## Python 模板

前提：`fixed_window` 仅在 `1 <= k <= len(nums)` 时有意义；`variable_window` 的“和不超过 `limit`”收缩逻辑要求 `nums` 非负、`limit >= 0`，且空窗口合法。窗口无效时每次移动左边界都会移除非负数，使窗口和不增并最终恢复合法，因此 `left` 最多到 `right + 1`。若含负数，窗口状态不再单调，应改用前缀和等方法。

```python
def fixed_window(nums, k):
    window_sum = 0
    best = float("-inf")

    for right, x in enumerate(nums):
        window_sum += x
        if right >= k:
            window_sum -= nums[right - k]
        if right >= k - 1:
            best = max(best, window_sum)

    return best
```

```python
def variable_window(nums, limit):
    left = 0
    total = 0
    best = 0

    for right, x in enumerate(nums):
        total += x
        while total > limit:
            total -= nums[left]
            left += 1
        best = max(best, right - left + 1)

    return best
```

## 讲解例题

### LeetCode 3. Longest Substring Without Repeating Characters

思路：窗口内不能有重复字符。用 `last_seen` 记录字符最近出现位置；遇到重复字符时，只把 `left` 移到重复位置右侧，并且不能回退。

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        last_seen = {}
        left = 0
        best = 0

        for right, ch in enumerate(s):
            if ch in last_seen and last_seen[ch] >= left:
                left = last_seen[ch] + 1
            last_seen[ch] = right
            best = max(best, right - left + 1)

        return best
```

复杂度：时间 `O(n)`，空间 `O(min(n, charset))`。

### LeetCode 209. Minimum Size Subarray Sum

思路：数组元素为正数，因此窗口和随着右边界扩张单调增加，随着左边界收缩单调减少。满足 `total >= target` 后尽量收缩，记录最短长度。

```python
from typing import List


class Solution:
    def minSubArrayLen(self, target: int, nums: List[int]) -> int:
        left = 0
        total = 0
        best = float("inf")

        for right, x in enumerate(nums):
            total += x
            while total >= target:
                best = min(best, right - left + 1)
                total -= nums[left]
                left += 1

        return 0 if best == float("inf") else best
```

复杂度：每个元素最多进出窗口一次，时间 `O(n)`，空间 `O(1)`。

## 常见坑

- 只有连续区间才能用滑动窗口；子序列通常不能。
- 窗口可收缩的前提通常是存在单调性，负数数组的“和不超过 K”不一定适合普通窗口。
- `while` 和 `if` 不可随便替换；需要持续收缩时必须用 `while`。
- 更新答案的位置要按题意区分：最长通常窗口合法后更新，最短通常满足条件时更新并收缩。
- 字符串窗口题要注意 `left` 不能回退。

## 面试表达

可以这样讲：我维护的是一个连续窗口，右指针负责纳入新元素，左指针负责恢复合法性。每个元素最多被左右指针访问一次，所以整体是线性的。
