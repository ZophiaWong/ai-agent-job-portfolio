# 面试资料总览

从目标岗位和当前弱项进入对应专题；每个专题都应按同一套
[练习协议](practice-protocol.md)完成，而不是只停留在阅读。

| 方向 | 入口 | 适用问题 |
| --- | --- | --- |
| AI Agent、RAG、评测与治理 | [AI 专项](01-AI/README.md) | 如何设计、评测和治理 Agent 系统？ |
| Python、数据库、服务端与可靠性 | [后端专项](02-后端/README.md) | 如何把 Agent 能力交付为可靠服务？ |
| 数据结构与算法 | [DSA 专项](03-DS_AL/README.md) | 如何在约束下写出可运行、可解释的解法？ |
| 简历、项目表达与行为面试 | [职业表达](04-career/README.md) | 如何用真实证据说明个人贡献和取舍？ |

## 使用方式

1. 先在[核心能力模型](../.codex/skills/interview-prep-coach/references/competency-model.md)中确定要证明的 C01–C13 能力。
2. 从上表进入一个专题，只选一个可验证的主题。
3. 按[练习协议](practice-protocol.md)完成独立作答、执行检查、迁移题和延迟复测。
4. 把真实项目主张回到[项目证据索引](../projects/README.md)核对，不把猜测当事实。
