## 使用规则

- 面试前从公开仓库核对当前实现，不依赖记忆猜测。
- 回答时区分 `implemented`、`planned`、`proposed improvement` 和 `unknown`。
- 私人的个人贡献、简历表述和模拟反馈保存在 `.local/interview-prep/projects.md`。
- 项目学习必须落到可观察能力，例如“解释审批状态机的重放安全”，而不是“看过 MeterDesk”。
