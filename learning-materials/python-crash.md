## 基础数据结构

### 基本数据结构的操作与区别

| 特性/操作    | 列表（list）                                                                  | 元组（tuple）             | 字典（dict）                                             | 集合（set）                                                                              |       |
| ------------ | ----------------------------------------------------------------------------- | ------------------------- | -------------------------------------------------------- | ---------------------------------------------------------------------------------------- | ----- |
| **有序**     | 是                                                                            | 是                        | 否（Python 3.7+ 按插入顺序保存）                         | 否                                                                                       |       |
| **可变**     | 是                                                                            | 否                        | 是                                                       | 是                                                                                       |       |
| **元素重复** | 允许                                                                          | 允许                      | 键不允许重复，值可以重复                                 | 不允许重复                                                                               |       |
| **访问方式** | 通过索引访问 (`lst[index]`)                                                   | 通过索引访问 (`t[index]`) | 通过键访问 (`d['key']`)                                  | 通过值判断 (`value in s`)                                                                |       |
| **切片操作** | 是                                                                            | 是                        | 否                                                       | 否                                                                                       |       |
| **常用操作** | `append()`, `insert()`, `remove()`, `pop()`, `sort()`, `reverse()`, `clear()` | `count()`, `index()`      | `get()`, `keys()`, `values()`, `items()`, `pop()`, `del` | `add()`, `remove()`, `discard()`, `clear()`, `union()`, `intersection()`, `difference()` |       |
| **判断元素** | `element in list`                                                             | `element in tuple`        | `key in dict`                                            | `element in set`                                                                         |       |
| **合并**     | `list1 + list2`                                                               | `tuple1 + tuple2`         | `dict1.update(dict2)`                                    | `set1                                                                                    | set2` |
| **获取长度** | `len(lst)`                                                                    | `len(t)`                  | `len(d)`                                                 | `len(s)`                                                                                 |       |
| **访问值**   | 通过索引 `lst[index]`                                                         | 通过索引 `t[index]`       | 通过键 `d[key]`                                          | 无法按索引访问                                                                           |       |
| **元素类型** | 支持任何数据类型                                                              | 支持任何数据类型          | 键必须是不可变类型（例如字符串、数字），值可以是任何类型 | 只能包含不可变类型（如数字、字符串、元组）                                               |       |
| **删除操作** | `remove()`, `pop()`, `del`, `clear()`                                         | 无法删除单个元素          | `del d[key]`, `pop()`, `clear()`                         | `remove()`, `discard()`, `clear()`                                                       |       |

### 简要总结

- 列表（list）：有序且可变，可以修改、添加、删除元素，支持重复元素，常用于需要存储多个有序元素的场景。
- 元组（tuple）：有序且不可变，不能修改内容，适合用于存储常量数据或需要保证数据不变的场景。
- 字典（dict）：存储键值对，键是唯一的，值可以重复，适合存储需要快速查找的键值数据。
- 集合（set）：无序且不可重复，适合存储需要去重的元素，支持并集、交集、差集等集合运算。
