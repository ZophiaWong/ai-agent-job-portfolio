## 3. 为什么不按 P0/P1 文件夹重排

如果按重要性重排，会适合短期冲刺，但不适合作为长期参考资料。Agent 系统学习有天然依赖关系：

```text
Agent 基础
  ↓
Workflow / Tool Calling
  ↓
RAG / Context / Memory
  ↓
Python 后端工程
  ↓
Evaluation / Security
  ↓
Project / Interview / Framework Lab
```

把 P0/P1 拆成不同文件夹，会破坏这个构建顺序。更稳妥的做法是主体按系统逻辑排列，每章头部标注岗位重要性。

第三轮掌握：07、13、references、CHANGELOG。
