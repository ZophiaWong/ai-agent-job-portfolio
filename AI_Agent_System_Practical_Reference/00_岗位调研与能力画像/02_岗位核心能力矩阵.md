# 岗位能力入口

本页不再维护另一套能力排名或数字分数。候选人能力、目标等级和证据标准统一引用[C01–C13 核心能力模型](../../.codex/skills/interview-prep-coach/references/competency-model.md)。

## 使用规则

- 始终引用能力模型里的稳定 ID；同义词不创建新能力。
- 阅读、打卡和自评不提升等级。等级需要独立作答、可执行代码、设计、项目或真实面试证据。
- 具体 JD 只是覆盖层：它可以调整 C01–C13 的优先级和目标等级，不改写已有证据，也不复制能力。

实际使用时，先在能力模型中选定缺口，再回到[手册索引](../00_README_学习路线与资料使用说明.md)找材料，用[2–3 周证据路线](../Part_04_项目与面试表达/13_总复习Checklist与学习计划.md)交付可检查产物。
