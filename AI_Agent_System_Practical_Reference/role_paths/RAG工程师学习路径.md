# RAG 工程师学习路径

> 生成日期：2026-04-29  
> 资料定位：AI Agent 应用岗位的系统学习 + 工程实战参考资料。  
> 来源基础：用户指定 GitHub 目录、公开 JD 调研、官方框架文档与经典论文。

## 1. 岗位定位

面向知识库问答、企业搜索、文档问答、代码库检索、金融/医疗/客服知识增强等岗位。核心是数据和检索质量。

## 4. 项目准备建议

准备一个多源知识库 RAG：PDF/Markdown/网页/代码文件接入，支持语义切块、Hybrid Search、Rerank、Citation、索引版本化和 RAG Eval。

## 5. 面试重点

Chunking 权衡；Embedding 选型；Hybrid Search 与 Rerank；Lost in the Middle；Graph RAG 适用场景；RAG 指标与 Bad Case 回流。
