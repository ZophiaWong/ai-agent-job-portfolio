# AI Agent 应用工程师学习路径

> 生成日期：2026-04-29  
> 资料定位：AI Agent 应用岗位的系统学习 + 工程实战参考资料。

> 注意：招聘信息会变化，岗位调研用于能力画像，不代表某公司当前仍在招聘。

## 1. 岗位定位

面向能把 LLM、RAG、工具调用、Workflow 和业务系统集成成可上线产品的应用工程师。重点不是训练大模型，而是系统集成、稳定性、评测、安全和业务指标。

## 2. 必须精读章节

01、02、03、04、05、08、09、10、11、14、16。

## 3. 建议掌握章节

06、07、12、13、15。

## 4. 项目准备建议

准备一个企业知识库 RAG Agent 或内部自动化 Agent：包含 RAG、LangGraph、Tool Calling、FastAPI、状态持久化、Trace、HITL 和降级。

## 5. 面试重点

Agent 与 Workflow 区别；LangGraph 为什么适合有状态任务；RAG 召回不准怎么优化；工具调用如何治理；Agent 成功率下降怎么排查。
